### Como correr o script de instalação

#### Clonar o repo.
```bash
git clone https://github.com/andresantosyt/teslamate.git
```

#### Entrar na pasta teslamate
```bash
cd teslamate
```

#### Tornar o ficheiro executavel
```bash
chmod +x install-teslamate.sh
```

#### Correr o script
```bash
./install-teslamate.sh
```
